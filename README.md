# 2026_03_A

舞鶴市の子ども人口と保育施設の関係を調査するプロジェクト（A班）

## ファイル構成

```
index.html                                    調査結果まとめページ
geo_correlation_report.md                     人口・保育施設の緯度経度相関分析レポート
reserch.txt                                   人口減少・保育施設利用率に関する考察メモ
262021_population_20230401.csv                舞鶴市 人口データ（2023/04/01時点）
262021_preschool_20230330.csv                 舞鶴市 保育施設データ
maizuru_child_population_heatmap_bw.png       子ども人口分布ヒートマップ
tile_cache/                                   地図タイルのキャッシュ
```

## ブランチ名規則

原則<種類>/<内容>という形でブランチを作成する

```
feature/add-login
fix/typo-homepage
docs/update-readme
```

## コミットメッセージ

原則Prefixをつける

```
feat: 新しい機能
fix: バグの修正
docs: ドキュメントのみの変更
style: 空白、フォーマット、セミコロン追加など
refactor: 仕様に影響がないコード改善(リファクタ)
perf: パフォーマンス向上関連
test: テスト関連
chore: ビルド、補助ツール、ライブラリ関連
```
