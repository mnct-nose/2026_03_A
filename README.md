# 2026_03_A

## ブランチ名規則

原則<種類>/<内容>という形でブランチを作成する

```
feature/add-login
fix/typo-homepage
docs/update-readme
```

## コミットメッセージ

原則Prefixをつける

```
feat: 新しい機能
fix: バグの修正
docs: ドキュメントのみの変更
style: 空白、フォーマット、セミコロン追加など
refactor: 仕様に影響がないコード改善(リファクタ)
perf: パフォーマンス向上関連
test: テスト関連
chore: ビルド、補助ツール、ライブラリ関連
```
